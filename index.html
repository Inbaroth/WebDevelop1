    Q&A


    Comparing between web 1.0 2.0 3.0:


    Comparing query results based on the Web generetion:
    The query: "Where can I buy costumes in Tel Aviv?"
    Web 1.0: probably we'll get static web sites with limited list of stores.
    In addition, most of the cases the results won't be accurate and it's better to search by given URL.
    Web 2.0: the results we'll be much more accurate and in high chances will contain sells web sites with changing content and opetions to "communicate" with the site by liking, chating etc.
    probably we'll get blogs and personal websites with recommendations, also we'll probably get talkbacks about the subject as results.
    Web 3.0: probably we'll get recommendations for stores selling costumes, and recommendations for websites selling costumes.
    Moreover, the results will come out way faster than in the previous generetions of the Web.
    In addition, probably the results we'll be close to other searches about the subject we ran before.
